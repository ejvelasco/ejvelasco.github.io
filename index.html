Front End

I strive to implement applications that look and feel great across devices. Anytime design is involved, I focus on creating elegant interfaces and a smooth user experience. I really like React and React Native.

Back End

I aim to develop APIs that are scalable, reliable, and mantainable. Most of the time, I work with NodeJS and the rich ecosystem of libraries that the NPM offers. I also like to work with Python. Working with awesome libraries like Scikit Learn and Tensorflow was what inspired me to create flareTrees, a small open source machine learning library for NodeJS.

Databases

To use, or not to use a relational data model? No problem! I am comfortable working with programs like MySQL or MongoDB. I have experience designing organized and efficient data models.
